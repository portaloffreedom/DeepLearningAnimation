CMake Qt5 GUI Template
----------------------

You need Qt5 and CMake 2.8.11 or higher

-- Requirements --
Compiler Requirements

- C++17 support

Library Requirements

- Eigen3
- Qt5
- Qt5::Charts
- Catch2 (for testing)

-- Build instructions --

cd /your/project/build
cmake ../
make

./nn_debug_view
